## Layout Goals:

- Home
- About Us (3 Pillars)
  - Hosting mixers
  - Brunch/Lunch and learns
  - Giving back to the community
- Events
- Gallery
- Register / Login (CRM) - PRIORITY
  - Standard Login
  - OAuth Login (Social Media)
  - What should the username format be? So that you can easily keep track of people
  - CRM should be linked to RSVP
  - Event Counter (how many times each person has attended the events)
  - Database that stores attendance
  - Anonymous RSVPing is allowed
- Contact Us
  - Name
  - Email
  - Phone
  - Message

Post-MVP Content: 
- Partners & Affiliates
  - Lewisville Chamber
  - Potentially volunteer organizations
- Testimonials (Success stories)

## Database Setup:

- SQL using Sequelize
- User Registration
  - E-mail blasts for events and networking opportunities
- Universal admin e-mail
  - Contact form submissions will be funneled to this account
